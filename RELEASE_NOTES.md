# Antica Fiamma 1.0.0

Release date: 22 July 2026

Live research environment: https://anticafiamma.it

Archived release DOI: https://doi.org/10.5281/zenodo.21479971

## Scope

Version 1.0.0 is the first citable, rights-safe archive of the Antica Fiamma public research environment. It freezes the public software shell, selected method-facing code, public documentation, runtime checks, deployment packaging, citation metadata, licence statements, and synthetic or redacted runtime samples. The live research environment remains continuously updated after this snapshot.

## Included

- public HTML shell, styles, client modules, static assets, Authority shell, and authority-page generator;
- public Guide, About, Interface Tour, Authority, and Fiamma research-room pages;
- selected build utilities and source-capture code retained for method transparency;
- public runtime checks and Cloudflare Pages packaging code;
- public repository, data-boundary, rights, technical, and maintenance documentation;
- rights-safe structural samples and aggregate coverage counts;
- README, CITATION.cff, Zenodo metadata, MIT licence, mixed-rights statement, and these release notes.

## Excluded

- complete Dartmouth Dante Project-derived commentary payloads and bulk commentary full text;
- generated author/work/personaggio pages that may cumulatively expose source-derived commentary text;
- private research materials and publication drafts;
- local snapshots, reports, caches, upload state, and deployment residue;
- secrets, tokens, credentials, and environment files;
- third-party source materials whose public redistribution has not been established.

## Rights

Project-owned software is released under MIT. Project documentation by Ruoci Song is released under CC BY 4.0 unless otherwise noted. Synthetic/redacted samples are documentation and do not convey rights in source materials. Antica Fiamma has written authorisation for current academic, non-profit website use of DDP-derived material; that authorisation is limited to the authorised website use and is not treated as a licence to redistribute the complete commentary corpus through GitHub, Zenodo, or another repository.
