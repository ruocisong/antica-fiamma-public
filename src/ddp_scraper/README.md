# Selected Source-Capture Utilities

This directory contains selected DDP source-capture and inspection utilities retained for lineage transparency.

Antica Fiamma is built with the Dartmouth Dante Project as its commentary source. These files help document part of the preparation history behind the public interface, but this directory is not a complete public data-release mechanism.

These utilities are not presented as a general invitation to re-harvest or
redistribute Dartmouth Dante Project materials. Anyone adapting them should
respect the Dartmouth Dante Project's own terms, attribution requirements,
server load, and permission boundaries.

## Public Role

The files here can help a technical reader understand:

- how source-facing records were inspected
- how commentary targets and line information were handled
- how early samples and record-fetching utilities were organized

## Boundary

This directory does not ship the full live runtime corpus.

The public repository also excludes the heavy frontend payloads under:

```text
demo/frontend/data/
```

Use this directory as method context, not as a complete reproduction package.

For the repository-level data and rights boundary, see
[`../../docs/public/data-boundary.md`](../../docs/public/data-boundary.md) and
[`../../docs/public/rights-and-permissions.md`](../../docs/public/rights-and-permissions.md).
