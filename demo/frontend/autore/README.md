# Static Autore Pages

This directory contains generated static author and work pages for the Antica Fiamma authority layer.

These pages mirror public authority data and make author/work paths linkable outside the main reading session.

## Public Role

The pages here support:

- author rooms
- work rooms
- source-facing authority navigation
- links from commentary cards into the authority layer
- public browsing outside the live line-reading state

## Generation

These pages are generated by:

```text
demo/build_authority_static_pages.py
```

They should be regenerated and published whenever author/work trees, counts, static copy, or authority routing changes.

## Deployment Note

Pages builds must include:

```text
autore/**
```

Otherwise `authority.html` can link to pages that never reach the live site.
