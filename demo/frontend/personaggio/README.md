# Static Personaggio Pages

This directory contains generated static personaggio pages for the Antica Fiamma authority layer.

Personaggio pages are separate from author and work rooms. A name can participate in more than one authority context, but those contexts should not be collapsed unless the data explicitly models that relation.

## Public Role

The pages here support:

- personaggio browsing
- links from commentary records into named figures
- distinction between author authority and represented figures
- public authority navigation outside the main reading session

## Generation

These pages are generated by:

```text
demo/build_authority_static_pages.py
```

Stale personaggio pages should be cleaned by the static build process, not left to drift.

## Deployment Note

Pages builds must include:

```text
personaggio/**
```
