## Runtime Data And Source-Derived Materials

The full runtime commentary payload used by the live Antica Fiamma interface
is not distributed through this repository. In particular, this repository
does not grant rights to redistribute Dartmouth Dante Project-derived
commentary texts or other source materials outside the terms set by their
source context.

The Dartmouth Dante Project remains the visible source archive for the
commentary layer. Antica Fiamma is an interface layer around that source, not
a replacement source archive.

## Sample Runtime Files

The files under [`sample_runtime/`](sample_runtime/) are a rights-safe sample
of the static asset structure. They are provided to document schemas,
publication boundaries, and build-time/runtime organization. They do not
include full Dartmouth Dante Project commentary texts.

apply to visible source materials.
